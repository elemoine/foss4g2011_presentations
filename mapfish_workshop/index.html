MapFish In Production

Goal of this workshop

Show one way to develop production-level MapFish applications.

Organization

Workshop document

  • http://mapfish/mapfish_workshop (in the vm)
  • http://www.mapfish.org/doc/tutorials/mapfishsample-workshop/index.html

Six modules

  1. Getting Started
  2. Understanding MapFish
  3. Installing and Running MapFishSample
  4. Studying MapFishSample
  5. Securing MapFishSample
  6. Going Further With Security

Modules include explanations and tasks.

Getting Started

Goal: setting up your working environment.

Understanding MapFish

Goal: a practical introduction to MapFish, to know what MapFish is and provides.

  1. Install MapFish
  2. Create a MapFish app
  3. Create a MapFish web service
  4. Understand the MapFish protocol

Installing and Running MapFishSample

Goal: know how to install and execute MapFishSample, and get to know about the tools we use for building, deployment, and execution.

  1. Download MapFishSample
  2. Build and Deploy MapFishSample
  3. Run MapFishSample
  4. Play with MapFishSample

http://demo.mapfish.org/mapfishsample/2.2

Studying MapFishSample

Goal: understand how MapFishSample is structured, and how it is connected to external components (MapServer, MapFish Print).

  1. General Structure
  2. Source Code Organization
  3. Modify The Client Code
  4. MapServer
  5. Print
  6. TileCache

Securing MapFishSample

Goal: show how to control access to MapFishSample services, and external services.

  1. Create User Model
  2. Set up Authentication
  3. Secure Application
  4. Secure MapFish Web Services
  5. Secure MapServer

Going Further With Security

http://www.mapfish.org/doc/tutorials/mapfishsample-workshop/en/going_further_with_security.html

  1. Extend User Model
  2. Secure POI Web Service
